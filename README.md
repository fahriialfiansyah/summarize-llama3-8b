# Llama 3 8B for Summarize Indonesian Text
A powerful text summarization tool utilizing the Llama 3 8B model. This project provides a notebook for generating concise and accurate summaries especially problems and solutions from large bodies of text, supporting a variety of input formats and customization options to meet diverse summarization needs
